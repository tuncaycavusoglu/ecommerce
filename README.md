# ShoeVault - Premium Ayakkabı Mağazası

Bu proje, React, TypeScript ve Tailwind CSS kullanılarak geliştirilmiş modern bir e-ticaret vitrin uygulamasıdır.

## 🚀 Projeyi Ayağa Kaldırma

Projeyi yerel ortamınızda çalıştırmak için aşağıdaki adımları izleyin:

### Gerekli Kurulumlar
- Bilgisayarınızda [Node.js](https://nodejs.org/) (önerilen sürüm: 18.x veya üzeri) kurulu olmalıdır.

### Adımlar

1. **Bağımlılıkları Yükleyin**
   Proje dizininde (bu dosyanın bulunduğu klasörde) terminali açıp şu komutu çalıştırın:
   ```bash
   npm install
   ```

2. **Geliştirme Sunucusunu Başlatın**
   Yükleme tamamlandıktan sonra projeyi başlatmak için:
   ```bash
   npm run dev
   ```

3. **Tarayıcıda Görüntüleyin**
   Sunucu başarıyla başladığında, tarayıcınızda şu adresi açarak uygulamayı görebilirsiniz:
   ```text
   http://localhost:7593
   ```

## 🛠 Kullanılan Teknolojiler
- **React (Vite):** Hızlı ve modern UI geliştirme.
- **TypeScript:** Güvenli ve ölçeklenebilir kod yazımı.
- **Tailwind CSS v3:** Temiz ve hızlı sayfa tasarımı (Light Theme).
- **React Router:** Sayfalar arası geçiş yönetimi.

*Not: Ürün ve vitrin verileri şimdilik `src/mocks/products.ts` dosyası üzerinden statik (mock) olarak sağlanmaktadır.*
